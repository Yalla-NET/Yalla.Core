Yalla.Core
==========

Yet Another Logging Library Abstraction for .NET (Core Library)

Supported Platforms
-------------------

* .NET Framework 4
* Silverlight 5
* Windows 8
* Windows Phone 8.1
* Windows Phone Silverlight 8
* Xamarin.Android
* Xamarin.iOS
* Portable Class Libraries
